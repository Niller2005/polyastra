## Next Steps for Users

1. **Restart bot** with updated code to populate Bayesian comparison data
2. **Run for 50-100 trades** with `BAYESIAN_CONFIDENCE=NO` (default)
3. **Compare methods** using:
   ```bash
   uv run python compare_bayesian_additive.py
   ```
4. **Verify data** using:
   ```bash
   uv run python check_bayesian_data.py
   ```
5. **Switch to Bayesian** if it shows superior performance:
   ```env
   BAYESIAN_CONFIDENCE=YES
   ```

## Notes

- Bayesian method is generally more conservative (e.g., 43.6% vs 3.1% on trade #529)
- Consider lowering `MIN_EDGE` slightly when switching to Bayesian
- All recent improvements (scale-in, price validation, multi-confirmation, evidence-based reversal) work with either method
- Quality factors applied to log-likelihood, not weighted scores (different from additive method)
